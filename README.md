# SpamScope v5.2 Pro+ (Yellow Edition)
> **Multi-threaded Discord automation tool for efficient messaging.**

This is a lightweight Python-based tool I developed to automate messaging across multiple Discord channels simultaneously. Originally created for personal use, I'm sharing it for anyone looking to save time and reduce manual typing.

###  Key Features:
* **Multi-Threading:** Supports 9 fixed channels + unlimited "Quick Add" channels.
* **Smart Timer:** Real-time countdown for every channel with a randomized `1-5s` offset.
* **Auto-Config:** Automatically saves your Token, Channel IDs, and Messages to `discord_config.json`.
* **Live Stats:** Track  Success and  Error counts directly on the dashboard.
* **Pause/Resume:** Change content on the fly without stopping the timers.

### Important Notes:
* **Display:** **Please maximize the window (Full-screen mode) to access all features and the full interface.**
* **Cooldown:** It is highly recommended to keep delays above 60s to ensure account safety.
* **Security:** Your Token is stored locally on your machine. No data collection.
